# nasa-api-app-example

A quick show-case of using golang and svelte as a secure data providing application

## Usage

**Backend:**

-requires a docker service running

```
cd src/backend
make start
```

**Frontend:**

- requires nodejs >=15

```
cd src/frontend
npm install
npm run serve
```

Example should be traversable at http://localhost:3000
